AuthService
service in module app.services

Description

Manages user registration, login, logout and user-logged checks

Methods

  • getUserFromToken()

    Locates middle part of JWT, which includes payload and extracts user from token

    Returns
    Object

    User object

  • isLoggedIn()

    Checks whether user exists, based on the existence of valid JWT token. Remember this doesn't say anything about status of the API.

    Returns
    Object

    A Promise

  • login()

    Handles http post to REST api to login existing user

    Returns
    Object

    A Promise

  • logout()

    Handles http post to REST api to logout existing user. This call invalidates JWT on server and deletes JWT from client local storage

    Returns
    Object

    A Promise

  • register()

    Handles http post to REST api to create new user

    Returns
    Object

    A Promise

  • urlBase64Decode(str)

    Decodes part of JWT token with payload

    Parameters
    ParamTypeDetails
    strString

    JWT token string

    Returns
    Object

    request config object
